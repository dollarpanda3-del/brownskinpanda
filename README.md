# AETHER // Cosmic Guardian

A self-contained browser prototype for a space-themed Earth science game where players monitor, scan, and protect Earth from orbit using NASA Earth observation-inspired systems.

## Run locally

```bash
npm start
```

Then open `http://localhost:4173`.

## What is in the prototype

- Orbital Earth viewport with animated atmosphere, clouds, weather glow, moon orbit, and satellite deployment
- Explorer, Mission, and Sandbox modes
- Scanable Earth regions with hazard, climate, and biosphere telemetry
- Time-lensing slider from 2026 to 2100
- Cosmic Guardian actions for scanning, intervention, rebalancing, and threat deflection
- NASA feed matrix mapped to real systems such as GIBS, EONET, EPIC, MODIS NDVI, IMERG, ECCO, AIRS, and OMI

## NASA references used for the concept

- GIBS: interactive global imagery for Earth exploration
- EONET v3: near real-time natural event metadata
- EPIC / DSCOVR: full-disk Earth imagery
- MODIS and NDVI: vegetation and land monitoring
- AIRS and OMI: atmospheric temperature, moisture, aerosols, and pollution
- ECCO: ocean circulation and climate state estimation
